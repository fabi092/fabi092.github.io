OpenGL Graphics and Shader Showcase

This project showcases the implementation of advanced Graphics and Shader Programming techniques using OpenGL. The scene depicts a rock column generated through the Marching Cubes algorithm, casting intricate shadows on a flat plane with the illusion of depth due to Depth Mapping. The dynamic shadowing system allows for a seamless switch between hard shadows, PCF, and VSM.

1. Marching Cubes Algorithm:
-Utilized GPU Look-Up Tables and Shaders to implement the Marching Cubes algorithm for dynamic generation of detailed rock formations.
-Created a variety of shapes within the rock structure, including outer boundaries, vertical columns, a central erasing column, planes, and a twisted form using procedural rules.
2. Tesselation:
-Applied Tesselation to dynamically adjust the vertex density based on distance.
-Utilized different levels of tessellation for inner and outer edges, enhancing the visual quality of the scene.
3. Parallax Mapping:
-Implemented a Parallax Shader to create the illusion of depth on a flat surface using a depth map.
-Achieved dynamic texture displacement to maintain the depth effect from various viewing angles.
4. Soft Shadows (VSM and PCF):
-Implemented Variance Shadow Mapping (VSM) using Gaussian distribution for shadow softness and better realism.
-Implemented Percentage Closer Filtering (PCF) for advanced shadow casting, allowing for a smoother transition between light and shadow.
